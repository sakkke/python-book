Pythonには様々な演算子があります。ここでは基本的な算術演算子を紹介します。

- **加算**: `+`
- **減算**: `-`
- **乗算**: `*`
- **除算**: `/`
- **整数除算**: `//`
- **剰余**: `%`
- **べき乗**: `**`

```python
# 加算
print(10 + 5)  # 15
# 減算
print(10 - 5)  # 5
# 乗算
print(10 * 5)  # 50
# 除算
print(10 / 5)  # 2.0
# 整数除算
print(10 // 3)  # 3
# 剰余
print(10 % 3)  # 1
# べき乗
print(2 ** 3)  # 8
```
